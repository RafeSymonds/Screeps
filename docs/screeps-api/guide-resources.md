# Resources Guide

## Energy
- Found in Sources (typically 2 per room)
- Harvested by WORK parts (2 energy/WORK/tick)
- Sources regenerate every 300 ticks (3000 energy capacity, or 1500 in reserved rooms)

## Minerals

Seven base types, one per room. Requires Extractor (RCL 6) + WORK parts.

### Base Compounds
- Hydroxide: H + O
- Zynthium Keanite: Z + K
- Utrium Lemergite: U + L
- Ghodium: ZK + UL

### Reactions
- Requires 3+ labs (2 input, 1 output)
- Labs must be within 2 squares of each other
- One mineral type per lab at a time

### Creep Boosts
- Cost: 30 mineral + 20 energy per body part
- Boosted parts function as 2-4x equivalent parts
- One compound per body part type

**Tier 1 Examples:**
- UH: +100% attack (ATTACK)
- UO: +200% harvest (WORK)
- KH: +50 carry capacity (CARRY)
- LO: +100% heal (HEAL)

Higher tiers (T2, T3 catalyzed compounds) provide stronger effects.

## Commodities

Four raw types in highway rooms: Metal, Silicon, Biomass, Mist (one per map quadrant).

### Factory Production (RCL 7+)
- **Compression**: Raw resources → bars (e.g., 500 Utrium + 200 Energy → 100 bars)
- **Battery**: 600 Energy → 50 batteries (portable energy)

### Production Chains (require Power Creep operators)
- Mechanical (Metal): Tube → Fixtures → Frame → Hydraulics → Machine
- Biological (Biomass): Phlegm → Tissue → Muscle → Organoid → Organism
- Electronical (Silicon): Switch → Transistor → Microchip → Circuit → Device
- Mystical (Mist): Concentrate → Extract → Spirit → Emanation → Essence

## Power
- Found in StructurePowerBank in highway rooms
- Destroy to collect dropped power
- Used for Power Creeps and GPL advancement
