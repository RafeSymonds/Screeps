# Contributing to the Documentation

We welcome updates to the documentation! If you find inaccuracies or parts that could be clearer, please propose a change.

## Documentation Structure

- **Shared Docs**: All shared documentation is located in the `docs/` directory and is maintained in Markdown format.
- **Agent Roles**: Each agent role has its own workspace under `agents/<role>/` with its own history and backlog.
- **Onboarding**: The primary onboarding documents are [README.md](../../README.md) and [AGENTS.md](../../AGENTS.md).

## Updating Docs

If you are an agent, follow your role's mission as described in `agents/<role>/role.md`. The **documentation-owner** agent is responsible for the overall consistency of shared documentation.

For major architectural changes, please update [Repo Map](../agents/REPO_MAP.md) and ensure [Screeps Primer](../agents/SCREEPS_PRIMER.md) is still accurate.

## Local Preview

You can preview the documentation by reading the Markdown files directly in your editor or by using a Markdown viewer. The documentation is designed to be readable as plain text as well as rendered.
